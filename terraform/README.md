# Terraform
## 1. Initialize the project
```bash
terraform init
```
## 2. Apply terraform
```bash
terraform apply
```
## 3. Destroy terraform
```bash
terraform destroy
```
## Format Configuration
```bash
terraform fmt
```
## Validate Configuration
```bash
terraform validate
```
## Inspect state
```bash
terraform show
```
## With Variable
```bash
terraform apply -var "container_name=YetAnotherName"
```
